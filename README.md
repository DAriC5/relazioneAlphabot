# relazioneAlphabot
Questa repository contiene il lavoro svolto durante le lezioni per il progetto dell'Alphabot:
1. la gestione dei sensori é stata fatta nella funzione forward della libreria prova
2. il server é generale per tutte le versioni del progetto
3. invece i client sono due:
  - la prima versione é basilare che rivece l'input da tastiera e lo invia al server
  - la seconda invece é gestita con pyinput e riceve l'input in base al tasto cliccato
